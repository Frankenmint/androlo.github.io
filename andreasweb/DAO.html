Andy's DDAO


This is a DDAO (Democratic Decentralized Autonomous Organization). It allows you to create a decentralized, democratically run organization. A DAO uses peer-to-peer communication for information sharing. The management and processing of information is handled by the Ethereum client. The Ethereum client makes sure that everyone's local data is synchronized with everyone elses. It also makes sure that all changes to the data (or the DAO itself) are done in accordance with the agreed-upon rules.

General DAO's can have many different rules. They do not even have to be democratically run. Rules are updated and kept current by the same mechanisms that processes the other information.

Andy's DDAO is based on Dennis McKinnons DOUG contract system.
